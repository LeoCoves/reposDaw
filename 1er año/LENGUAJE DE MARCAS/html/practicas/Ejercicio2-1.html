Bienvenidos al módulo de Lenguajes de Marcas

Los contenidos de este ejercicio tratan de ilustrar el funcionamiento de los elementos básicos de un documento HTML.

Encabezados

Los encabezados tienen como función estructurar el contenido del documento en partes para identificar los apartados del texto. Normalmente, se utiliza un encabezado de nivel 1, que corresponde con el mayor tamaño, para identificar los apartados principales. Se pueden ir utilizando otros encabezadks de menor tamaño para indicar los subapartados. Existen 6 tipos de encabezados

Lineas horizontales

Suelen utilizarse las lineas hroizontales para introducir una separacion visual en el texto del documento.
